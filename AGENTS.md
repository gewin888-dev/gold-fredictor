# 黄金走势实时监控与预测系统

## 目标

构建一个 Python 系统，自动采集黄金相关宏观、利率、美元、期货持仓、央行买金、新闻情绪、中国黄金溢价等数据，并生成黄金多空评分、风险提示和飞书机器人推送。

## 开发原则

1. 先做可运行 MVP。
2. 优先使用官方免费 API。
3. 需要授权的数据源只预留接口。
4. 所有密钥放到 `.env`。
5. 所有数据记录 `source`、`timestamp`、`updated_at`。
6. 每次修改后说明运行方法。
7. 不直接给出投资建议；仅提供数据分析、黄金多空评分、风险提示和决策参考。

## 第一阶段功能

- FRED 数据采集
- SQLite 入库
- 黄金多空评分
- Streamlit 仪表盘
- 飞书机器人推送接口预留
